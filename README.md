# React 電子商務 Mongodb Expressjs

> 使用MERN 和 Redux建立的電子商務平台

**[Demo 點我](https://www.malshop.mlclmtan.ga)**

![screenshot](https://github.com/mlclmtan/malcolmshop/blob/master/uploads/1.png)

## 功能

- 全功能購物車
- 商品評價
- 熱賣商品幻燈片
- 商品分頁
- 商品搜尋
- 訂單與個人賬戶
- 管理員商品管理
- 管理員賬戶管理
- 管理員訂單管理
- 標記商品已送達
- 下訂單流程（物流、付款方式、等）
- 付款方式PayPal / 信用卡整合
